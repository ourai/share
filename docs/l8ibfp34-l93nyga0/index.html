接口

数据总览

  • 「热点问答榜」接口
  • 「热点知识榜」接口
    • 前端复用知识引擎首页的「热门排行榜」接口
  • 「实时政策更新」接口
    • 前端复用知识引擎首页的「实时政策更新」接口
  • 「试点地区满意度」接口
    • 要支持图表显示
  • 其他统计指标接口
    • 通过一个接口返回所有指标数据
    • 每个指标的数据结构要一致
    • 「内容总条数」要支持图表显示

知识库概览

  • 「热门知识」接口
  • 「知识库动态」接口
    • 前端复用「运维-知识库历史记录」列表接口
  • 「知识库详情」接口
    • 前端复用知识库编辑时的详情接口，并且后端要加上关联的可选标签数据（仅限一级）
  • 统计指标接口
    • 通过一个接口返回所有指标数据
    • 每个指标的数据结构要一致

详情页指标

  • 知识详情指标接口
    • 返回数据要根据指标类别分组
